Markdown Reference

Syntax supported in merMDitor — standard CommonMark plus extended plugins.

Headings

Use # symbols at the start of a line. One # is H1, up to six ###### for H6.

SyntaxOutput
# Heading 1Heading 1
## Heading 2Heading 2
### Heading 3Heading 3
#### Heading 4Heading 4
